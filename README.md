# Customer-Transaction-Analysis
Customer transaction data analysis using Python and EDA
# Objective
Analyze customer transaction data to identify spending patterns, trends, and anomalies.

# Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn

# Key Insights
- Identified top spending categories
- Analyzed monthly transaction trends
- Detected high-value customers
- Found unusual transaction patterns

# Project Structure
- transactions.csv → dataset
- analysis.ipynb → analysis code

# Outcome
This project demonstrates my ability to clean, analyze, and visualize data to generate business insights.
